

Usage:
- open pycharm project
- Need to install packages: flask and pymysql

OR

- pip3 install virtualenv (test using [virtualenv --version])
- virtualenv venv 
- source venv/bin/activate (or [venv\Scripts\activate] in windows)
- python3 app.py
